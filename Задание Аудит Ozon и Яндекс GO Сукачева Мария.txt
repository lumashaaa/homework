1. OZON
1) Включила VoiceOver в настройках универсального доступа айфона. 
2) Вышла на домашний экран, где нажала дважды кнопку поиск, чтобы найти приложение Ozon. 
3) Зашла в приложение двойным тапом по иконке.
4) Внутри приложения с помощью движения пальца по экрану нашла окно поиска, дважды нажала на него, после чего смогла ввести название «Кружка для чая». 
5) С помощью свайпов смогла выбрать товар, дважды нажав на него, попала в карточку товара. 
6) С помощью медленного движения по экрану можно узнать стоимость, цвет товара. 
7) Не удалось выйти обратно в меню поиска - стрелочку «назад» VoiceOver не видит и не озвучивает. 
8) VoiceOver не видит крестика при показе всплывающей рекламы - невозможно закрыть окно.
9) Нижняя панель навигации (Домой, Поиск, Ozon Банк, Избранное, Корзина, Профиль) недоступна, так как VoiceOver не озвучивает названия вкладок, говорит только что «на вкладке», фокус не перемещается между элементами.

2. Яндекс GO
1) Открыла приложение Yandex GO с включённым VoiceOver.
2) С помощью перемещения пальца по экрану нашла поле для ввода адреса, двойным тапом активировала и успешно ввела текст.
3) Удалось выбрать адрес из списка с помощью свайпов.
4) Можно настроить параметры поездки (например, комментарий водителю) с помощью стандартных VoiceOver свайпов.
5) Стрелками можно вернуться на домашний экран.
6) Можно зайти в меню и изучить его содержимое: добавить адреса, посмотреть историю поездок, обратиться в поддержку. Все элементы озвучиваются корректно и доступны для навигации.
7) Основной недочёт: выбор тарифа поездки недоступен. VoiceOver не фокусируется на элементах тарифа, невозможно переключиться между вариантами ( «Эконом», «Комфорт» и т.д.).
